Etapas para realizar este projeto;
Na máquina local:
  - abrir prompt comando (cmd)
  - procurar a pasta desejada que contenha os arquivos para o push
  - digitar os comandos nesta ordem;
    
     - git init  (para gerar inicialização e gerar o arquivo .git)
     - git add . (para adcionar e reconhecer qualquer mudança que tenha sido feita nos arquivos ou pasta)
     - git commit -m main  "um texto referente a este trabalho" (para comitar)
     - git remote add origin https..........(endereço do repositório)
     - git push -u origin main (para enviar para servidor GitHub)
 No site do GitHub
     atualizar a pagina e confirmar o repositório atualiazado.
